### 如何使用

1. **启动项目**：
   ```bash
   # 后端
   cd server
   go run .
   
   # 前端
   cd web
   npm run serve
   ```

2. **访问前端页面**：
   - 登录系统
   - 访问：学生课程关联管理页面
   - 使用批量选课和批量添加学生功能

3. **系统功能**：
   - 基础的学生课程关联管理
   - 学生可以选修多门课程
   - 课程可以被多个学生选修
   - 管理学生课程的成绩
   - 管理学生课程的状态（已选课/已完成/已退课）

这个实现满足了常见教务系统的基本需求，包括选课、成绩管理和课程状态追踪。你可以根据实际需求进一步扩展功能。
